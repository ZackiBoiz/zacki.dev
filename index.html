xnand

A fun little Talkomatic bot built for Talkomatic Classic, Talkomatic Beta (Dev), and Talkomatic Modern. Has many features like games, moderation, and chat engagement!

rplace

Another bot built for Talkomatic. This bot allows users to place emojis on a 20x20 grid with each other to try to create works of art! Originally created by Test Account, now modified and maintained by me.

Watchdog

A bot built for Talkomatic Watchparty. It includes features such as loading and playing videos, seeking, customizing speed, and syncing videos for users.

GitHub

pixel-grid

I've created a project similar to r/place. In this canvas, users can draw around an infinite grid. I've built this with help from cheezburger0! Includes fun functions like a rich content chat with emoticons.

ZackiBot

A piano-playing bot built for Multiplayer Piano on a variety of clones! It includes features like playing MIDIs, queuing, and fun miscellaneous interactions.

Minebot

This is a Discord bot used for scraping information and providing an interactable client on Cavegame.io, a 2D multiplayer survival game. The bot can access live information and statistics like player counts, game states, and chat logs. It also includes a client that in-game users can use for chat engagement.

HolyC Thingy
